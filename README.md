# Agentic Honeypot API

An AI-powered Agentic Honeypot that detects scam intent and autonomously engages scammers to extract actionable intelligence such as UPI IDs, bank details, phishing links, and phone numbers without revealing detection.

## Features
- Scam intent detection with confidence scoring
- Autonomous multi-turn scam engagement
- Structured intelligence extraction
- Secure API key authentication
- Public cloud deployment

## Tech Stack
- Python, FastAPI
- OpenAPI 3.1
- Render (Cloud Deployment)
